가볍고 빠른 Python FastAPI + Redis 기반 오픈소스 URL 단축 서비스.

## LRU란?
**LRU**는 길고 복잡한 URL을 반대로 짧고 간결한 URL 즉 **LRU**로 바꾸어 주는 서비스 입니다.

URL의 반대 = LRU
